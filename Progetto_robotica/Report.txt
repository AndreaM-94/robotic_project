Aprile 22
Abbiamo scelto il progetto basato su “Livox-MD-360”.  Passi preliminari:
    1. Installazione di ubuntu 22.04
    2. Installazione di ROS 2 humble
    3. Installazione del sofware LIVOX viewer 2
Collegamento e Setup del sensore lidar
Sudo apt install net-tools
bisogna avviare la connessione con il sensore, quindi bisogna cercare la porta utilizzata dall’ethernet, con il comando:
lshw -class network 



Bisogna inserire il nome logico in :
sudo ifconfig “nome_logico” 192.168.1.50

Download livox viewer2
www.livoxtech.com/mid-360/downloads
Scaricare Livox Viewer 2 per linux, estrarre in una cartella il pacchetto e utilizzare il seguente comando:
./livox_viewer_2.sh.
Sostituire il nome se è diverso.

bisogna settare il workspace di ROS2, quindi riferirsi a questa guida:
https://docs.ros.org/en/foxy/Tutorials/Beginner-Client-Libraries/Creating-A-Workspace/Creating-A-Workspace.html
1) Assicurarsi che tutti i requisiti siano installati
(colcon sicuramente manca, leggere il link seguente per ulteriori informazioni)
sudo apt install python3-colcon-common-extensions
https://docs.ros.org/en/foxy/Tutorials/Beginner-Client-Libraries/Colcon-Tutorial.html

2)SCARICARE GTSAM
2.1) AGGIUNGERE LA REPO 
sudo add-apt-repository ppa:borglab/gtsam-release-4.1
2.2) INSTALLARE LO GTSAM
sudo apt update
sudo apt install libgtsam-dev libgtsam-unstable-dev


2) Creare la cartella del workspace
mkdir -p ~/ros2_ws/src

3) utilizzare questo comando per copiare la repository di github nel workspace creato(richiesto anche nella nostra repository)
git clone https://github.com/rajvishnu07/lio_sam_mid360 src/LIO_SAM_MID360

4) utilizzare il comando:
cd ~/ros2_ws/src/LIO_SAM_MID360
colcon build --symlink-install
source install/setup.bash
